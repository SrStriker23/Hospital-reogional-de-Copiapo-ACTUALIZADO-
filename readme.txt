Instalación de Librerías

Para que este proyecto funcione correctamente, necesitas instalar las siguientes librerías:

Copiar código
pip install pymysql
pip install pandas
pip install tkcalendar
Configuración de la Base de Datos

Crea una base de datos llamada hospital.
Importa el archivo que contiene todas las tablas y datos de la base de datos en hospital.
Nota: Al crear la base de datos, asegúrate de que el cotejamiento sea utf8mb4_general_ci si estás usando phpMyAdmin.

Si tu base de datos tiene contraseña, agrégala en el archivo ConsultasSQL en la línea 8.

Uso del Código

1-.Descarga el archivo ZIP del proyecto.

2-.Extrae la carpeta y colócala en el escritorio.

3-.Abre el archivo Hospital-copiapo.py.

4-.En la línea 550, realiza los siguientes cambios:

	Reemplaza la ruta C:\Users\vicen\Desktop\Hospital-Regional-Copiapo-main con la ruta correspondiente en tu computadora.

	Cambia C y vicen por la letra de la unidad de disco y el nombre de usuario de tu computadora.

5-.Usa las herramientas de búsqueda y reemplazo de Visual Studio Code para asegurarte de que todos los cambios se realicen correctamente.

Nota Adicional

Asegúrate de que la escala de tu PC esté configurada al 100% para que las ventanas se puedan visualizar completamente.
